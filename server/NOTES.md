# Server scaffold - notes

This is a minimal NestJS scaffold intended to get the API started. It does not implement authentication, database persistence or robust error handling yet — those will be added in the MVS phase.

Commands (inside server/):

1. Install deps (dev only):
   npm install
2. Dev run (requires Nest CLI):
   npm run start:dev
3. Build:
   npm run build
4. Start (production):
   npm run start

Local with Docker Compose (recommended):
   docker-compose up --build

# 🔊 Ses Efektleri Klasörü

Bu klasörde uygulama ses efektleri bulunur.

## Gerekli Ses Dosyaları

Aşağıdaki MP3 dosyalarını bu klasöre ekleyin:

| Dosya Adı | Açıklama | Süre (öneri) |
|-----------|----------|--------------|
| `success.mp3` | Başarılı işlem sesi | 0.5-1 sn |
| `error.mp3` | Hata/Red sesi | 0.5-1 sn |
| `notification.mp3` | Bildirim sesi | 0.5 sn |
| `pdf_generated.mp3` | PDF oluşturma sesi | 1-1.5 sn |
| `scan_beep.mp3` | Barkod tarama sesi | 0.2 sn |
| `click.mp3` | Buton tıklama sesi | 0.1 sn |
| `save_success.mp3` | Kayıt başarılı sesi | 0.8 sn |
| `delete.mp3` | Silme/İptal sesi | 0.5 sn |

## Ses Dosyası Önerileri

### Ücretsiz Ses Kaynakları:
1. **mixkit.co** - Ücretsiz ses efektleri
2. **freesound.org** - CC0 sesler
3. **zapsplat.com** - Ücretsiz sesler

### Ses Özellikleri:
- Format: MP3
- Sample Rate: 44.1kHz
- Bitrate: 128-192 kbps
- Mono veya Stereo

### Ses Profilleri:

**Başarı Sesi:**
- Yüksek tonlu, neşeli
- Örnek: "ding", "tada", "success chime"

**Hata Sesi:**
- Düşük tonlu, ciddi
- Örnek: "buzz", "error buzzer", "fail"

**Bildirim Sesi:**
- Orta tonlu, dikkat çekici
- Örnek: "pop", "notification", "alert"

**PDF Sesi:**
- Kağıt/ yazıcı teması
- Örnek: "page flip", "printer", "document"

**Tarama Sesi:**
- Kısa, keskin
- Örnek: "beep", "scan", "barcode"

## Örnek Ses Dosyası İsimleri (Mixkit)

- `mixkit-software-interface-start-2574.mp3` → success.mp3
- `mixkit-wrong-answer-fail-notification-946.mp3` → error.mp3
- `mixkit-sci-fi-positive-notification-266.mp3` → notification.mp3
- `mixkit-paper-slide-1531.mp3` → pdf_generated.mp3
- `mixkit-game-ball-tap-2073.mp3` → scan_beep.mp3
- `mixkit-mouse-click-close-1113.mp3` → click.mp3

## Not

Ses dosyaları olmadan uygulama çalışmaya devam eder. 
Ses efektleri otomatik olarak devre dışı kalır.
